SQL优化总结(持续更新...)

2019.03.12

SQL是影响程序性能的一个环节，知道常见的SQL优化方法有必要的

  • 使用=来比较，不使用<>

    = 增加了索引的使用几率

  • 使用LIMIT 1来返回一条记录

    使用LIMIT 1避免全表扫描

  • 避免使用SELECT *,这样会全表扫描

    SELECT * 会全表扫描

  • WHERE子句中列尽量被索引

    加快查询速度

  • 使用LIMIT来完成分页

    分页工具有很多，底层都是LIMIT去实现的
